Control Structure

Control structures are used to control the flow of the program. It is used to execute the code based on certain conditions. The control structures in Python are:

1. Conditional Statements

Conditional statements are used to execute the code based on certain conditions. The conditional statements in Python are:

a. If statement

The if statement is used to execute the code if a certain condition is true. The syntax of the if statement is:

Python
            
if condition:
    # code to be executed if the condition is true

Example:

Python
            
x = 10
if x > 5:

    print("x is greater than 5")

b. If-else statement

The if-else statement is used to execute the code based on the condition. If the condition is true, the code inside the if block is executed, and if the condition is false, the code inside the else block is executed. The syntax of the if-else statement is:

Python
            
if condition:
    # code to be executed if the condition is true
else:
    # code to be executed if the condition is false

Example:

Python
            
x = 10
if x > 5:
    print("x is greater than 5")
else:
    print("x is less than or equal to 5")

c. If-elif-else statement

The if-elif-else statement is used to execute the code based on multiple conditions. If the condition in the if block is true, the code inside the if block is executed. If the condition in the elif block is true, the code inside the elif block is executed. If none of the conditions are true, the code inside the else block is executed. The syntax of the if-elif-else statement is:

Python
            
if condition1:
    # code to be executed if condition1 is true
elif condition2:
    # code to be executed if condition2 is true
else:
    # code to be executed if none of the conditions are true

Example:

Python
            
x = 10
if x > 5:
    print("x is greater than 5")
elif x == 5:
    print("x is equal to 5")
else:
    print("x is less than 5")

2. Looping Statements

Looping statements are used to execute a block of code multiple times. The looping statements in Python are:

a. For loop

The for loop is used to iterate over a sequence (list, tuple, string, etc.) and execute the code for each item in the sequence. The syntax of the for loop is:

Python
            
for item in sequence:
    # code to be executed for each item in the sequence

Example:

Python
            
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

b. While loop

The while loop is used to execute a block of code as long as a certain condition is true. The syntax of the while loop is:

Python
            
while condition:
    # code to be executed as long as the condition is true

Example:

Python
            
i = 1 
while i <= 5:
    print(i)
    i += 1

3. Control Statements

Control statements are used to control the flow of the program. The control statements in Python are:

a. Break statement

The break statement is used to exit the loop prematurely. It is used to break out of the loop when a certain condition is met. The break statement can be used in both for and while loops. The syntax of the break statement is:

Python
            
for item in sequence:
    if condition:
        break

Example:

Python
            
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    if fruit == "banana":
        break
    print(fruit)

b. Continue statement

The continue statement is used to skip the rest of the code inside the loop and continue with the next iteration. It is used to skip the current iteration of the loop when a certain condition is met. The continue statement can be used in both for and while loops. The syntax of the continue statement is:

Python
            
for item in sequence:
    if condition:
        continue

Example:

Python
            
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    if fruit == "banana":
        continue
    print(fruit)

c. Pass statement

The pass statement is used as a placeholder when a statement is required syntactically but you do not want any code to execute. The pass statement is a null operation; nothing happens when it is executed. The syntax of the pass statement is:

Python
            
for item in sequence:
    pass

while condition:
    pass

Example:

Python
            
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    pass
