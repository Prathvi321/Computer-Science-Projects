Variables & Data Types

Variables are used to store data values in Python. They can be assigned different data types, such as integers, floats, strings, booleans, and NoneType. Here's an example:

  • Primitive Types: These are the fundamental data types
    • int: Integer values (e.g., 5, -3, 0)
    • float: Floating-point values (e.g., 3.14, -0.5, 2.0)
    • str: String values (e.g., "Hello", 'Python', "123")
    • bool: Boolean values (True or False)
    • NoneType: None value (represents the absence of a value)
  • Collections: These are data structures that hold collections of data.
    • List: Ordered, mutable sequences of items. You can change lists after they
      • Python
                        
numbers = [1, 2, 3, 4]
numbers.append(5) # Adds 5 to the end of the list
numbers[0] = 10 # Changes the first element to 10
print(numbers) # Output: [10, 2, 3, 4, 5]
    • Tuple: Ordered, immutable sequences of items. Tuples cannot be changed after creation. They are often used for representing fixed collections of data, like coordinates.
      • Python
                          
coordinates = (3, 5)
# coordinates[0] = 10  # This would cause an error because tuples are immutable
print(coordinates[1]) # Output: 5
    • Dictionary: Unordered collections of key-value pairs. Each key is unique and maps to a value.
      • Python
                          
person = {"name": "Bob", "age": 30, "city": "New York"}
print(person["name"])  # Output: Bob
person["age"] = 31   # Modifying a value
person["country"] = "USA" # Adding a new key-value pair
print(person) # Output: {'name': 'Bob', 'age': 31, 'city': 'New York', 'country': 'USA'}
    • Set: Unordered collections of unique items. Sets do not allow duplicate elements.
      • Python
                          
unique_numbers = {1, 2, 2, 3}  # Becomes {1, 2, 3}
unique_numbers.add(4)
print(unique_numbers) # Output: {1, 2, 3, 4} (order may vary)

Operators

Operators are used to perform operations on variables and values. Python supports various types of operators, including arithmetic, comparison, logical, assignment, and bitwise operators.

• Arithmetic Operators: These operators are used to perform mathematical operations like addition, subtraction, multiplication, division, and more. Here are some examples:

Python
            
# Addition
a = 5
b = 3
print(a + b)  # Output: 8

# Subtraction
print(a - b)  # Output: 2

# Multiplication
print(a * b)  # Output: 15

# Division
print(a / b)  # Output: 1.6666666666666667

# Floor Division
print(a // b)  # Output: 1

# Modulus
print(a % b)  # Output: 2

# Exponentiation
print(a ** b)  # Output: 125

• Comparison Operators: (greater than), < (less than), >= (greater than or equal to), <= (less than or equal to), == (equal1 to), != (not equal to). These operators return boolean values..

Python
            
x = 5
y = 3

print(x > y)  # Output: True
print(x < y)  # Output: False
print(x >= y)  # Output: True
print(x <= y)  # Output: False
print(x == y)  # Output: False
print(x != y)  # Output: True

• Logical Operators: These operators are used to combine conditional statements. The logical operators in Python are and, or, and not.

Python
            
x = 5
print(x > 0 and x < 10)  # Output: True
print(x > 0 or x < 3)  # Output: True
print(not(x > 0 and x < 10))  # Output: False

• Assignment Operators: These operators are used to assign values to variables. For example, x = 5 assigns the value 5 to the variable x.

Python
            
x = 5
x += 3  # Equivalent to x = x + 3
print(x)  # Output: 8

y = 10
y -= 2  # Equivalent to y = y - 2
print(y)  # Output: 8

• Identity Operators: is, is not. Used to check if two variables refer to the same object in memory.

Python
            
x = [1, 2, 3]
y = [1, 2, 3]
z = x

print(x is y)  # Output: False
print(x is z)  # Output: True
print(x is not y)  # Output: True

• Membership Operators: in, not in. Used to check if a value is present in a sequence (e.g., a list, tuple, or string).

Python
            
numbers = [1, 2, 3, 4, 5]

print(3 in numbers)  # Output: True
print(6 in numbers)  # Output: False
print(3 not in numbers)  # Output: False
print(6 not in numbers)  # Output: True

• Bitwise Operators: These operators perform bitwise operations on integers. They treat numbers as sequences of bits (0s and 1s) and perform operations bit by bit.

Python
            
a = 60  # 60 = 0011 1100
b = 13  # 13 = 0000 1101

print(a & b)  # Output: 12 (0000 1100)
print(a | b)  # Output: 61 (0011 1101)
print(a ^ b)  # Output: 49 (0011 0001)
print(~a)  # Output: -61 (1100 0011)
print(a << 2)  # Output: 240 (1111 0000)
print(a >> 2)  # Output: 15 (0000 1111)

• Type Conversion: Python allows you to convert one data type to another. For example, you can convert an integer to a float, a float to an integer, or a string to an integer.

Python
            
x = 5
y = 3.14
z = "10"

print(float(x))  # Output: 5.0
print(int(y))  # Output: 3
print(int(z))  # Output: 10

Input and Output:

Input: The input() function is used to take user input in Python. It prompts the user to enter a value and returns the entered value as a string.

Python
            
name = input("Enter your name: ")
print("Hello, " + name + "!")  # Output: Hello, [name]!

Comments:

• Single-line comments: Comments are used to explain code and make it more readable. In Python, you can use the # symbol to add comments to your code. Anything after the # symbol on a line is considered a comment and is ignored by the Python interpreter.

Python
            
# This is a comment
print("Hello, World!")  # This is another comment

• Multi-line comments: You can use triple quotes (''' or """) to create multi-line comments in Python. This is useful when you need to add longer comments that span multiple lines.

Python
            
'''
This is a multi-line comment.
It can span multiple lines.
'''
print("Hello, World!")

Indentation:

Indentation is used to define the structure of your code in Python. Blocks of code are defined by their indentation level. You can use spaces or tabs to indent your code, but it's important to be consistent. Indentation is used to indicate the beginning and end of loops, functions, classes, and other code blocks.

Python
            
if x > 0:
    print("x is positive")
else:
    print("x is negative")
