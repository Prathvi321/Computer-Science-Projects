Html (Hypertext Markup Language):It's the standard markup language for creating web pages. HTNL Elements are the building block of HTML page.

Syntax:Html uses tags to define elements. Tags are enclosed in angle brackets <> and usually come in pairs - an opening tag and a closing tag. The closing tag has a / before the tag name like </tagname>.

Basic Structure:

Html
            
<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <!-- content goes here-->
</body>
</html>

Tags:

  • <!DOCTYPE html>: Defines the document type and version of HTML.
  • <html>: Contains the entire HTML document.
  • <head>: Contains meta-information about the document, such as title, links to stylesheets, scripts, etc.
  • <title>: Sets the title of the document, which appears in the browser tab.
  • <body>: Contains the content of the document, such as text, images, links, etc.
  • <!-- comment -->: Used to add comments to the code. Comments are not displayed in the browser.
  • <h1> to <h6>: Headings of different sizes, with h1 being the largest and h6 the smallest.
  • <p>: Paragraph tag for text content.
  • <a>: Anchor tag for links. The href attribute specifies the URL of the page the link goes to.
  • <img>: Image tag for displaying images. The src attribute specifies the image file.
  • <alt>: Specifies an alternative text for an image, if the image cannot be displayed.
  • <ul> <ol> <li>: Used for creating lists. <ul> creates an unordered list, <ol> creates an ordered list, and <li> defines a list item.
  • <div>: Division or section in an HTML document. Used for grouping and styling content.
  • <span>: Inline container used to apply styles to a section of text.
  • <style>: Contains CSS rules for styling the HTML document.
  • <script>: Contains JavaScript code or links to external scripts.

Tables:

Tables are used to display data in rows and columns. The basic structure of a table consists of the <table> element to create the table, <tr> for table rows, <th> for table headers, and <td> for table data cells.

  • <table>: Creates a table.
  • <tr>: Defines a row in the table.
  • <th>: Defines a header cell in the table.
  • <td>: Defines a data cell in the table.
  • <caption>: Provide title for a table or Adds a caption to the table.
  • Table Example:
    • Monthly Sales Report
    Month Sales
    January 1000
    February 1500

    Html
                    
<table>
    <caption>Monthly Sales Report</caption>
    <tr>
        <th>Month</th>
        <th>Sales</th>
    </tr>
    <tr>
        <td>January</td>
        <td>1000</td>
    </tr>
    <tr>
        <td>February</td>
        <td>1500</td>
    </tr>
</table>

Text Formatting:

HTML provides various tags to format text, such as <b> for bold, <i> for italic, <u> for underline, <mark> for highlighting, <sub> for subscript, and <sup> for superscript.

  • <b>: Bold text
  • <i>: Italic text
  • <u>: Underlined text
  • <mark>: Highlighted text
  • <sub>: Subscript text
  • <sup>: Superscript text
  • <s>: Strikethrough text

Links and Navigation:

Links are created using the <a> tag, with the href attribute specifying the URL of the page the link goes to. Links can be styled using CSS to change their appearance.

  • <a href="url">Link Text</a>: Creates a hyperlink with the specified URL and link text.
    • <a href="#section">Link to Section</a>: Creates an internal link to a section within the same page.
    • <a href="mailto:email">Email Link</a>: Creates a link to send an email to the specified email address.
    • <a href="tel:phone">Phone Link</a>: Creates a link to call the specified phone number.
  • <nav>: Defines a navigation link.
  • <ul> & <li>: Often used to Creates an unordered list of navigation Menu Links.

Attributes:

Attributes provide additional information about HTML elements. They are always included in the opening tag and are written as name-value pairs, separated by an equals sign.

  • id: Specifies a unique id for an element.
  • class: Specifies one or more class names for an element (used for styling with CSS).
  • style: Specifies inline CSS styles for an element.
  • title: Specifies extra information about an element (displayed as a tooltip).
  • href: Specifies the URL of the page the link goes to.
  • src: Specifies the URL of an image or other media file.
  • alt: Specifies an alternative text for an image, if the image cannot be displayed.
  • width & height: Specifies the width and height of an element.
  • target: Specifies where to open the linked document.

Forms:

Forms are used to collect user input, such as text fields, checkboxes, radio buttons, and submit buttons. The <form> element is used to create a form, and various input elements are used to collect data.

  • <form>: Creates a form for user input.
  • <input>: Creates an input field for the user to enter data.
  • <textarea>: Creates a multiline text input field.
  • <button>: Creates a clickable button.
  • <select> & <option>: Creates a dropdown list for the user to select from.
  • <label>: Defines a label for an input element.
  • <fieldset> & <legend>: Groups related elements in a form and provides a caption for the group.

Images:

Images are displayed using the <img> tag, with the src attribute specifying the URL of the image file. The alt attribute provides alternative text for the image, which is displayed if the image cannot be loaded.

  • <img src="url" alt="description">: Displays an image with the specified URL and description.
  • <figure> & <figcaption>: Groups an image and its caption together.

Audio and Video:

Audio and video files can be embedded in an HTML document using the <audio> and <video> tags. The src attribute specifies the URL of the media file, and the controls attribute adds playback controls.

  • <audio src="url" controls>: Embeds an audio file with playback controls.
  • <video src="url" controls>: Embeds a video file with playback controls.
  • <source>: Specifies multiple sources for audio or video files, allowing the browser to choose the most suitable one.

Meta Tags:

Meta tags provide metadata about the HTML document, such as the character set, author, description, and keywords. They are placed in the <head> section of the document.

  • <meta charset="UTF-8">: Specifies the character encoding for the document.
  • <meta name="description" content="description">: Provides a description of the document.
  • <meta name="keywords" content="keywords">: Specifies keywords related to the document.
  • <meta name="author" content="author">: Specifies the author of the document.
  • <meta http-equiv="refresh" content="5;url=page.html">: Redirects to another page after 5 seconds.

Lists:

HTML provides tags to create ordered lists (<ol>), unordered lists (<ul>), and list items (<li>). Lists can be nested inside one another to create sublists.

  • <ol>: Creates an ordered list with numbered items.
  • <ul>: Creates an unordered list with bulleted items.
  • <li>: Defines a list item in an ordered or unordered list.

Comments:

Comments in HTML are used to add notes or explanations to the code. They are not displayed in the browser and are only visible in the source code.

  • <!-- comment -->: Adds a comment to the code.
